筆記：Event listener的兩種使用方式：
  • insert into html tag
    • 在 html 檔案： <button onclick="func(this)">點擊這裡</button>
    在 js 檔案： func(element){...};

  • use AddEventListener("attribute", function(){...})
    • 在 html 檔案： <button id="btn">點擊這裡</button>
    在 js 檔案： document.getElementById("btn").addEventListener("click", function(){...})

  • 兩者的差別：在html上屬性多加了on，在js則維持原意 => html:"onclick" / js:"click"
